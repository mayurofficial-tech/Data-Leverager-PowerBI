# 📊 Data Leverager - Power BI ETL Project

## 📌 Description
Data Leverager is a Power BI ETL (Extract, Transform, Load) project that demonstrates how to collect, clean, transform, and integrate data from multiple sources using Power Query. The project builds a high-quality, analysis-ready dataset without using DAX, focusing entirely on Power Query transformations.

## 🎯 Objective
The objective of this project is to design a scalable ETL pipeline that integrates multiple data sources, performs data cleaning, and prepares structured data for analysis and reporting.

## 📂 Data Sources
- Excel files (Monthly Sales Data)  
- Employee dataset (Employees.xlsx)  
- Web data (HTML table source)  

## 🚀 Features
- Multi-source data integration  
- Data cleaning and preprocessing  
- Removal of duplicates, null values, and inconsistencies  
- Data type conversion and formatting  
- Feature engineering (profit calculation, age calculation, time-based columns)  
- Pivot and unpivot transformations  
- Data merging using joins  
- Automated data refresh using parameters  

## 🛠️ Technologies Used
- Microsoft Power BI  
- Power Query (M Language)  
- Microsoft Excel  
- Web Data Source

## ▶️ How to Use
1. Install Microsoft Power BI Desktop  
2. Open the `.pbix` file  
3. Update file paths if required  
4. Click on Refresh to load and transform data
